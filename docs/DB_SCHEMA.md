# DB_SCHEMA

Основний сайт не має локальної схеми бази даних.

Якщо зміни стосуються окремого API або адмінки, схему треба документувати в межах відповідного підпроєкту.

## dentis-site-api D1

Міграція `0007_news_i18n.sql` додає англомовні поля для новин:

- `news`: `badge_en`, `title_en`, `desc_en`, `date_en`.
- `scheduled_articles`: `badge_en`, `title_en`, `desc_en`.

Поля `title_en` і `desc_en` є обов'язковими для показу запису в `/api/public/news?lang=en`; якщо вони порожні, запис пропускається в EN-відповіді.
