# PROJECT_CONTEXT

Сайт `https://dentis.kr.ua/` для стоматологічної клініки Dentis у Кропивницькому.

Стек: Vite, React, TypeScript, Tailwind CSS, `vite-react-ssg`, PWA. Основна мова інтерфейсу — українська, є англійські маршрути.

Пріоритети: швидкий mobile-first перший екран, SEO, стабільний hero-блок, акуратне збереження кириличного контенту.
