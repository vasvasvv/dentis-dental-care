# DESIGN_RULES

Зберігати преміальний, спокійний візуальний стиль Dentis: темний hero, gold accents, чиста типографіка.

Hero-відео є частиною першого враження і має залишатися, але не повинно блокувати LCP на мобільних.

Не змінювати тексти, кольори, layout або анімації під час технічних performance-фіксів без окремого запиту.
